# MusicPlayer-in-Qt-Plus

基于Qt的音乐播放器的设计与实现

## 项目运行

### 运行环境
win10

### 编辑工具
Qt Creator 9.0.1

### 编译器
MinGW 8.1.0

## 功能和画饼
- [x] 基本的UI框架
- [x] 在线音乐搜索
- [x] 音乐播放
- [x] 歌词显示及滚动
- [ ] 本地音乐播放
- [ ] 我喜欢的音乐和歌单
- [ ] 本地数据存储
- [ ] 云端数据存储
